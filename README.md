# google-disk-adder
Easily adds disks to Google Compute Engines

## Usage
```
cd /tmp
wget https://raw.githubusercontent.com/jamrizzi/google-disk-adder/master/add-disk.sh
sudo chmod +x add-disk.sh
sudo ./add-disk.sh
```
